## 三、类型约束

ScalaUpperLowerBounds/ImplicitClass

[T <: xx] =》T都是xx的子类或者说xx就T的上界（Upper Bounds） 类似 java <T extends xx>
[T >: xx] =》T都是xx的父类或者说xx就T的下界（lower bounds）类似 java <T super D>
